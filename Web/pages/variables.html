JavaScript Variables

Variables are containers for storing data.

In JavaScript, there are 3 ways to declare a variable:

  • var
  • let
  • const

1. The var keyword

The var keyword was used in all JavaScript code from 1995 to 2015.

Variables declared with var have function scope.

var x = 5;
var y = 6;
var z = x + y;
console.log(z);

2. The let keyword

The let keyword was introduced in ES6 (2015).

Variables declared with let have block scope.

They cannot be redeclared in the same scope.

let x = "Hello";
// let x = "World"; // This will cause an error
x = "World"; // This is allowed (reassignment)
console.log(x);

3. The const keyword

The const keyword was also introduced in ES6.

Variables defined with const cannot be redeclared and cannot be reassigned.

const PI = 3.14159;
// PI = 3.14; // This will cause an error
console.log(PI);
Tip: Always use const if the value should not change. Use let if you can't use const. Only use var if you MUST support very old browsers.
