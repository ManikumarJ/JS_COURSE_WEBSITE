JavaScript Operators

JavaScript operators are used to perform operations on values.

Arithmetic Operators

Arithmetic operators are used to perform arithmetic on numbers:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (Remainder)
  • ++ Increment
  • -- Decrement
Example
let x = 10;
let y = 5;
let z = x + y;
console.log("Addition: " + z);
console.log("Modulus: " + (x % 3)); // 1

Assignment Operators

Assignment operators assign values to variables: =, +=, -=, etc.

Comparison Operators

Comparison operators result in true or false.

  • == equal to
  • === equal value and equal type
  • != not equal
  • !== not equal value or not equal type
  • > greater than
  • < less than
let a = 5;
let b = "5";
console.log(a == b);  // true
console.log(a === b); // false (different types)

Logical Operators

Logical operators are used to determine the logic between variables or values.

  • && and
  • || or
  • ! not
